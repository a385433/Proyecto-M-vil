# Nanys Care — Mobile (scaffold)

Scaffold inicial de la aplicación móvil con Expo + TypeScript.

Requisitos:
- Node.js (16+ recomendable) y npm o yarn
- Expo CLI (opcional): `npm install -g expo-cli`

Instalación:

```bash
cd mobile-app
npm install
# o
# yarn install
```

Ejecutar:

```bash
npm run start
# o
npm run android
npm run ios
```

Contenido inicial:
- `src/App.tsx`: Punto de entrada con navegación.
- `src/screens/*`: Pantallas básicas (Login, Home, Search, Profile, Reservation).
